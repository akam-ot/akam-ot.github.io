Machine Learning, NLP, Hugging Face

Fine-tuned xlm-roberta-base for Named Entity Recognition in Kurmanji Kurdish. Trained on 7,919 annotated sentences to detect PER, LOC, and ORG entities. Achieved an F1 score of 0.87 on the held-out test set.

Kurdish Dialect Classifier

Machine Learning, NLP, Gradio, Hugging Face

A lightweight web application that identifies whether a given Kurdish text (in Arabic script) is written in Sorani or Kurmanji (Badini). Built with Gradio for the UI and trained a Logistic Regression classifier using TF-IDF vectorization. Achieved 92% classification accuracy on test data.

Python, Flask, Regex, Serverless

A web tool that converts Kurdish text between Arabic and Latin scripts. Built with Flask and deployed as a serverless API on Vercel, it handles linguistic nuances like bizroke insertion for accurate transliteration.
